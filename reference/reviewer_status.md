# Summarise reviewers' progression of an article

This function summarises the status of reviewers who are willing to
review for a particular article.

## Usage

``` r
list_reviewers(article)

reviewer_status(article)
```

## Arguments

- article:

  Article id, like `"2014-01"`

## Examples

``` r
if (FALSE) list_reviewers("Submissions/2020-114") # \dontrun{}
if (FALSE) { # \dontrun{
reviewer_status("Submissions/2020-114")
} # }
```
