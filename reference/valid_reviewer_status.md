# A list of all valid reviewer statuses.

A list of all valid reviewer statuses.

## Usage

``` r
valid_reviewer_status
```

## Format

An object of class `character` of length 9.

## Examples

``` r
valid_reviewer_status
#> [1] "invited"   "agreed"    "declined"  "abandoned" "revision"  "minor"    
#> [7] "major"     "reject"    "accept"   
```
