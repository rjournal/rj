# A list of all valid statuses.

A list of all valid statuses.

## Usage

``` r
valid_status
```

## Format

An object of class `character` of length 26.

## Examples

``` r
valid_status
#>  [1] "submitted"             "acknowledged"          "passed initial checks"
#>  [4] "resubmission"          "needs reviewers"       "needs editor"         
#>  [7] "updated"               "out for review"        "major revision"       
#> [10] "minor revision"        "revision received"     "accepted"             
#> [13] "copy edited"           "online"                "proofed"              
#> [16] "out for proofing"      "style checked"         "with AE"              
#> [19] "AE: major revision"    "AE: minor revision"    "AE: accept"           
#> [22] "AE: reject"            "reject and resubmit"   "published"            
#> [25] "withdrawn"             "rejected"             
```
